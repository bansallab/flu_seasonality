# Human Behavior and Built Environments Drive Influenza Seasonality in the US

This repository contains the code (and processed data, to be added) supporting the manuscript:

**Pullano G, Tiu A, Srebric J, Milton DK, Marr LC, Bansal S.**  
*Human Behavior and Built Environments Drive Influenza Seasonality in the US.*


This repository provides:

- Stochastic age-structured transmission model code
- Multi-variate regression analysis code
